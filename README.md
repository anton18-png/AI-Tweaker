**GPT Windows 11 Optimizer** — это инструмент для автоматизированного подбора, тестирования и генерации твиков (оптимизаций) Windows 11 с помощью искусственного интеллекта (ChatGPT/GPT-4). Программа автоматически применяет твики, сравнивает производительность системы до и после, и сохраняет лучшие решения.

## Возможности

- **Автоматический бэкап реестра** перед началом работы.
- **Бенчмаркинг**:  
  - Копирование большого объёма файлов (имитация нагрузки на диск).
  - Открытие браузера и измерение времени запуска.
  - Открытие случайного .reg-файла из `C:\Backup` в Notepad с замером времени полной загрузки текста.
- **Автоматический подбор твиков**:  
  - Использует GPT-4 для выбора и генерации оптимальных .reg/.bat/.cmd твиков из папки `Brian`.
  - Генерирует как твики, так и скрипты для их отката.
- **Безопасность**:  
  - После каждого твика сравнивает производительность, при ухудшении — автоматически откатывает изменения.
- **Сохранение лучших твиков**:  
  - Автоматически собирает и сохраняет самые эффективные твики в отдельные файлы.
- **Гибкость**:  
  - Количество итераций можно задать через аргумент командной строки (`-i`).

## Структура проекта

```
├── main.py                # Главный скрипт
├── requirements.txt       # Зависимости Python
├── settings.ini           # Настройки бенчмарков (создается автоматически)
├── README.md              # Документация
├── optimizer.log          # Лог файл
├── Backup/                # Папка с бэкапами реестра
├── Brian/                 # Папка с твиками (.reg, .bat, .cmd)
├── elevator.exe           # Вспомогательные утилиты для повышения прав
├── launcher.exe
├── PowerRun.exe
├── icon.ico               # Иконка для exe
├── dist.bat               # Батник для сборки в exe
└── ...                    # Прочие служебные файлы
```

## Генерируемые файлы

- `win11_optimized_X.reg` - Твики для оптимизации
- `win11_undo_X.reg` - Твики для отката изменений
- `win11_optimized_X.bat` - Скрипты оптимизации
- `win11_undo_X.bat` - Скрипты отката
- `strongest_tweak.reg` - Лучшие найденные твики
- `strongest_tweak.bat` - Лучшие найденные скрипты

## Установка

1. **Установите Python 3.8+**  
2. **Установите зависимости:**
   ```
   pip install -r requirements.txt
   ```
3. **Распаковка архива** `Brian.7z`
4. **(Опционально) Соберите exe:**
   ```
   dist.bat
   ```
   или вручную:
   ```
   pyinstaller --onefile --icon=icon.ico --add-data "elevator.exe;." --add-data "launcher.exe;." --add-data "PowerRun.exe;." --add-data "Brian;Brian" main.py
   ```

## Использование

### Базовое использование
```bash
python main.py
```

### Параметры командной строки
```bash
python main.py -i 3                    # 3 итерации оптимизации
python main.py --without_backup        # Без создания бэкапа
python main.py -i 5 --without_backup   # Комбинирование параметров
```

1. **Запустите скрипт:**
   ```
   python main.py -i 5
   ```
   где `-i` — количество итераций оптимизации (по умолчанию 5).

3. **Что происходит:**
   - Создаётся бэкап реестра в `C:\Backup`.
   - Производится эталонный бенчмарк (копирование, браузер, notepad).
   - GPT-4 выбирает и генерирует твики, применяет их, делает бенчмарк, сравнивает результаты.
   - При ухудшении — твик откатывается.
   - В конце сохраняются лучшие твики.

4. **Результаты:**
   - Таблица с результатами оптимизации.
   - Файлы `strongest_tweak.reg` и `strongest_tweak.bat` — самые эффективные твики.

## Настройка дополнительных бенчмарков

Создайте файл `settings.ini` для настройки дополнительных тестов:

```ini
[Benchmark]
# URL для тестирования браузера
browser_urls = https://google.com, https://youtube.com

# Программы для тестирования (формат: название|путь|аргументы)
test_programs = 
    Notepad|C:\Windows\System32\notepad.exe|
    Calculator|C:\Windows\System32\calc.exe|
    Paint|C:\Windows\System32\mspaint.exe|

# Команды для тестирования (формат: название|команда|аргументы)
test_commands = 
    Dir|dir|C:\
    System Info|systeminfo|
    Task List|tasklist|

# Таймауты (в секундах)
browser_timeout = 30
program_timeout = 10
command_timeout = 10

# Включение/отключение бенчмарков
enable_browser_benchmark = true
enable_program_benchmark = true
enable_command_benchmark = true

# Настройки логирования
log_level = INFO
log_file = optimizer.log
```

## Бенчмарки

Программа выполняет следующие тесты:

### Основные бенчмарки
- **Копирование файлов**: Тест скорости диска
- **Открытие браузера**: Тест запуска приложений
- **Открытие Notepad**: Тест работы с файлами

### Дополнительные бенчмарки (через settings.ini)
- **Программы**: Тестирование запуска указанных приложений
- **Команды**: Выполнение системных команд
- **URL**: Тестирование открытия веб-страниц

## Логирование

Все действия записываются в `optimizer.log`:
- Информация о системе
- Результаты бенчмарков
- Генерируемые твики
- Ошибки и предупреждения

## Безопасность

- Автоматическое создание бэкапов реестра
- Откат изменений при ухудшении производительности
- Проверка результатов перед применением
- Логирование всех операций
